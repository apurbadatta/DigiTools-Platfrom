# 🚀 DigiTools - Premium Digital Marketplace

A modern, fast, and responsive digital product marketplace platform built with **React** and **Tailwind CSS**. This project allows users to browse premium digital tools, add them to a dynamic shopping cart, and manage their purchases seamlessly.

---

## 🛠️ Technologies Used

This project leverages the power of modern web development tools:

- **Frontend Framework:** [React.js] (Hooks, Functional Components)
- **Styling:** [Tailwind CSS] (For modern and responsive UI)
- **Icons:** [React Icons](FontAwesome)
- **Data Handling:** JSON
- **State Management:** React `useState` & `useEffect` (Prop Drilling/State Lifting)

---

## ✨ Key Features

### 1. Dynamic Shopping Cart System

Users can add products to their cart with a single click. The system includes **State Lifting** logic, ensuring the cart count is updated in real-time across the **Navbar** and the **Product Section**. It also prevents duplicate items from being added.

### 2. Tab-Based Navigation (Products vs. Cart)

A smooth, interactive toggle system that allows users to switch between the **Product Grid** and the **Shopping Cart View** without reloading the page. This provides a single-page application (SPA) experience.

### 3. Interactive UI & Success Alerts

The project features a highly polished UI with hover effects, responsive grids, and **Instant Notifications**. Whether adding a product, removing an item, or checking out, users receive immediate feedback for their actions.
